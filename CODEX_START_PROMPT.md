# VS Code / Codex 初回プロンプト

このリポジトリは、Street Fighter 6のキャラクター別コンボ・セットプレイをスマートフォンから確認するためのGitHub Pages用静的サイトです。

最初に `AGENTS.md` と `README.md` を読み、次に `assets/css/site.css`、`assets/js/character-page.js`、`characters/luke/data/meta.js`、各カテゴリーのデータファイル、`templates/combo-entry.template.js` を確認してください。

- フレームワーク・外部ライブラリ・ビルド処理は使わない
- 共通の見た目と描画処理は `assets/` に置く
- キャラクター固有データは `characters/<character-id>/data/` のカテゴリー別ファイルへ保存する
- 共通描画処理へキャラクター固有情報を直接書かない
- コンボ手順は `steps` 配列で管理し、HTMLの矢印やカード要素は描画側で生成する
- PC・スマホともにコンボカードは1列の横長表示にする
- 正式名称は「セットプレイ」、内部IDは `setplay`
- 現在はカテゴリー絞り込みを使った1ページ表示を維持する
- 情報増加時は同じカテゴリー別データを利用して表示ページを分割する

まず現在の実装がこの方針に沿うかを確認し、問題があれば変更前に「問題点」「変更対象」「修正案」を提示してください。実装後は変更ファイルと確認結果を報告してください。
